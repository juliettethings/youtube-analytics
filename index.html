🔑 YouTube Data API 키 발급 방법
  1. Google Cloud Console 접속 → 프로젝트 생성 (또는 기존 선택)
  2. 좌측 메뉴 → API 및 서비스 → 라이브러리
  3. "YouTube Data API v3" 검색 → 사용 설정
  4. 좌측 메뉴 → 사용자 인증 정보 → + 사용자 인증 정보 만들기 → API 키
  5. 생성된 키 복사 → 여기에 붙여넣기
✅ 무료 · 일 10,000 쿼터 (채널 100개 이상 분석 가능)
✦ Anthropic (Claude) API 키 발급 방법
  1. console.anthropic.com 접속 → 회원가입 / 로그인
  2. 좌측 메뉴 → API Keys → Create Key
  3. 키 이름 입력 후 생성 → 복사 → 여기에 붙여넣기
💡 댓글 분석 1회 기준 약 $0.003 (Haiku) ~ $0.015 (Sonnet) 수준
